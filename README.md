
1. Αδειοδότηση
Η Άδεια MIT είναι η πιο δημοφιλής, σύντομη & επιτρεπτική επιλογή, επιτρέποντας απεριόριστη χρήση. Η συγκεκριμένη άδεια επιτρέπει την ελεύθερη χρήση, τροποποίηση & διανομή του κώδικα από οποιονδήποτε, αρκεί να αναφέρεται ο αρχικός δημιουργός. Με αυτόν τον τρόπο διασφαλίζεται ότι ο κώδικας παραμένει ανοικτός & μπορεί να χρησιμοποιηθεί ή να βελτιωθεί χωρίς περιορισμούς. Η άδεια MIT είναι απλή και κατανοητή.

2. Επιχειρηματικό Μοντέλο – Managed Support
Η υπηρεσία αυτή μπορεί να χρησιμοποιηθεί για την παροχή υπηρεσιών “managed support” υποστήριξης WordPress σε μικρές επιχειρήσεις ή ιδιώτες. Μέσω του εργαλείου γραμμής εντολών, ο τεχνικός μπορεί να ελέγχει βασικές πληροφορίες του WordPress site, όπως τα τελευταία άρθρα, χωρίς να χρειάζεται είσοδο στο διαχειριστικό περιβάλλον. Η υπηρεσία μπορεί να προσφέρεται με μηνιαία συνδρομή & να περιλαμβάνει βασικό τεχνικό έλεγχο, έλεγχο λειτουργίας του site & απλή υποστήριξη σε περίπτωση προβλημάτων. Παρόλο που το εργαλείο είναι ανοικτού κώδικα και δωρεάν, η αξία της υπηρεσίας προκύπτει από τη γνώση & την τεχνική υποστήριξη που παρέχει ο επαγγελματίας στον πελάτη.

3. Διαφάνεια
Η δημοσίευση του κώδικα στο GitHub αυξάνει τη διαφάνεια & την αξιοπιστία του εργαλείου. Οποιοσδήποτε μπορεί να δει πώς λειτουργεί, να ελέγξει τον κώδικα & να προτείνει βελτιώσεις. Παράλληλα, μειώνεται ο κίνδυνος λαθών ή κακής χρήσης, καθώς ο κώδικας είναι ανοικτός & ελέγξιμος. Η διαφάνεια αυτή ενισχύει την εμπιστοσύνη των χρηστών & ακολουθεί τη φιλοσοφία του ανοικτού λογισμικού.

Δημιουργία εικονικού περιβάλλοντος: python3 -m venv venv

Ενεργοποίηση εικονικού περιβάλλοντος: source venv/bin/activate

Εγκατάσταση βιβλιοθήκης: pip install requests

Δημιουργία αρχείου εξαρτήσεων: pip freeze > requirements.txt

Απενεργοποίηση εικονικού περιβάλλοντος: deactivate
